Instapups

Instapups is a web application inspired by Instagram for people to post photos of their puppies and check out photos of friends and other users. The app was built with Ruby on Rails on the backend, a PostgreSQL database and React.js with a Redux framework on the frontend. Some features include: user signup/login, infinite scroll loading of photo feed, image likes/comments and user follows.

Color Swap

Color Swap is a browser game inspired by the mobile version, Color Switch. The goal of this game is to beat the highest score by tapping the ball and matching the colors as it passes through each obstacle. The obstacles have a variety of shapes, are made up of four different colors and rotate constantly to make it difficult for the user to match the colors.

jCurry

jCurry is a library to simplify DOM manipulation inspired by jQuery. Some of the features included in this library are abilities to traverse and manipulate elements, handle events and simplify HTTP requests.
